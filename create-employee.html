Create Employee

The Create Employee form allows you to add new employees to the system with all necessary information, including personal details, employment information, and role assignments.

Accessing Create Employee

  1. Log in to the system with appropriate credentials
  2. Navigate to the Branch Dashboard
  3. Click on "Employees" in the sidebar menu under the Organization section
  4. Click the "Create Employee" button (the button may appear as text, an icon, or similar depending on your interface) on the All Employees page
  5. The Create Employee form will load with default values

Form Fields Overview

The Create Employee form contains several sections with required and optional fields:

Basic Information:

  • Department - (dropdown) Select the department the employee will belong to (required)
  • Employee Number - (auto-generated) Unique identifier for the employee (automatically generated based on branch code)
  • Full Name - (text input) Enter the employee's full name (required)
  • Email - (email input) Enter the employee's email address (required, must be unique)
  • Phone - (text input) Enter the employee's phone number (optional)
  • Address - (textarea) Enter the employee's address (optional)

Personal Information:

  • Date of Birth - (date picker) Select the employee's date of birth (optional)
  • Gender - (dropdown) Select the employee's gender (optional)
  • Nationality - (dropdown) Select the employee's nationality (defaults to Nigerian)
  • Emergency Contact Name - (text input) Enter emergency contact name (optional)
  • Emergency Contact Phone - (text input) Enter emergency contact phone (optional)

Employment Information:

  • Hire Date - (date picker) Select the employee's hire date (required, defaults to today)
  • Termination Date - (date picker) Select the employee's termination date (optional)
  • Status - (dropdown) Select the employee's status (active, inactive, terminated, on probation, on leave)
  • Probation End Date - (date picker) Select when probation ends (optional)
  • Shift Preference - (dropdown) Select preferred shift type (morning, afternoon, night, rotating, flexible)

Compensation Information:

  • Salary - (number input) Enter the employee's annual salary (optional)
  • Hourly Rate - (number input) Enter the employee's hourly rate (optional)
  • Tax ID - (text input) Enter the employee's tax identification number (optional)
  • Bank Account - (text input) Enter the employee's bank account number (optional)

Additional Information:

  • Allergies - (textarea) Enter any known allergies (optional)
  • Profile Photo - (file upload) Upload the employee's profile photo (optional, max 2MB)
  • Last Performance Review Date - (date picker) Enter last review date (optional)
  • Performance Rating - (number input) Enter initial performance rating (optional, 0-5 scale)

Role Assignment:

  • Roles - (checkboxes) Select roles to assign to the employee (optional)

Creating an Employee

Follow these steps to create a new employee:

Step-by-Step Process:

  1. Select the appropriate department from the dropdown (required)
  2. Verify the auto-generated employee number or note it for reference
  3. Fill in the required fields: Full Name and Email
  4. Complete other fields as needed for your organization
  5. Upload a profile photo if available (optional)
  6. Select appropriate roles for the employee (optional)
  7. Click the "Save Employee" button (the button may appear as text, an icon, or similar depending on your interface)

Approval Workflow:

The creation process differs based on your user role:

For Super Administrators:
  • Employee is created immediately in the system
  • No approval required
  • Employee receives default login credentials
For Regular Employees:
  • A modal appears asking for a reason for the creation
  • Enter a detailed reason (minimum 5 characters required)
  • Click "Submit" to create an approval request
  • The request is sent to super administrators for review and approval

Creating Missing Dependencies

If needed, you can create related records directly from the form:

Creating a New Department:

  1. If the required department doesn't exist in the dropdown
  2. Click the "Create Department" button (the button may appear as text, an icon, or similar depending on your interface) next to the department dropdown
  3. A modal will appear with department creation fields
  4. Fill in the department details and save
  5. The new department will be available in the dropdown

Creating a New Branch:

  1. If the required branch doesn't exist in the dropdown
  2. Click the "Create Branch" button (the button may appear as text, an icon, or similar depending on your interface) next to the branch dropdown
  3. A modal will appear with branch creation fields
  4. Fill in the branch details and save
  5. The new branch will be available in the dropdown

Validation and Error Handling

The form validates all required fields before submission:

Required Field Validation:

  • Department must be selected
  • Employee number must be unique
  • Name is required
  • Email is required and must be unique
  • Hire date is required
  • Status is required

Format Validation:

  • Email must be in valid email format
  • Phone number format is validated
  • Profile photo must be an image file (JPG, PNG, etc.) and under 2MB
  • Numeric fields accept only valid numbers

Tips and Best Practices

  • Always verify that the employee number is unique before creating
  • Use consistent naming conventions for employee names
  • Ensure email addresses are unique across the system
  • Assign appropriate roles based on job responsibilities
  • Keep employee information up-to-date
  • Upload a profile photo for better identification
  • Consider data privacy regulations when collecting personal information
