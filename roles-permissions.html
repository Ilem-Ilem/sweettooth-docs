Roles & Permissions Management

The Roles & Permissions section allows super administrators to manage user roles and their associated permissions within the system.

Accessing Roles & Permissions

  1. Log in to the system with super administrator credentials
  2. Navigate to the Branch Dashboard
  3. Click on "Roles" in the sidebar menu (only visible to super admins)

Viewing Existing Roles

The main roles page displays a table of all existing roles with the following columns:

  • # - Role ID number
  • Role Name - The name of the role
  • Guard - The authentication guard for the role
  • Created At - Date the role was created
  • Actions - Available actions for the role

Creating a New Role

  1. Click the "Add New Role" button (the button may appear as text, an icon like "+", or similar depending on your interface)
  2. In the role creation modal that opens, fill in the following details:
    • Role Name - Enter a unique name for the role
    • Guard - Select the appropriate guard (typically "web")
  3. Select the permissions to assign to this role using the checkboxes
  4. Click "Save" to create the role

Note: Clicking the close button (typically an "X" icon) in the top-right corner of any modal will close it without saving changes.

Editing an Existing Role

  1. Find the role you want to edit in the roles table
  2. Click the "Edit" button (the button may appear as text, a pencil icon, or similar depending on your interface) in the Actions column for that role
  3. The role editing modal will open with current values pre-filled
  4. Modify the role name or permissions as needed
  5. Click "Save" to update the role

Viewing Role Permissions

  1. Find the role you want to inspect in the roles table
  2. Click the "View Permissions" button (the button may appear as text, an eye icon, or similar depending on your interface) in the Actions column
  3. A modal will appear showing all permissions assigned to that role
  4. Click the close button (typically an "X" icon) in the top-right corner to close the modal

Deleting a Role

  1. Find the role you want to delete in the roles table
  2. Click the "Delete" button (the button may appear as text, a trash icon, or similar depending on your interface) in the Actions column
  3. A confirmation dialog will appear asking if you're sure you want to delete the role
  4. Click "Confirm" to delete the role or "Cancel" to keep it

Warning: Deleting a role will remove it from all users assigned to that role. This action cannot be undone.

Bulk Operations

You can perform actions on multiple roles at once:

  • Select multiple roles using the checkboxes in the first column of the table
  • Choose an action from the "Bulk Actions" dropdown (e.g., "Delete Selected")
  • Confirm the action when prompted

Creating New Permissions

When creating or editing a role, you can also create new permissions:

  1. While in the role creation/editing modal, look for the "Add Permission" option
  2. Click the "Create Permission" button
  3. Fill in the permission name and guard in the new modal that appears
  4. Click "Save" to create the new permission
