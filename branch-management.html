Branch Management

The Branch Management section allows super administrators to manage all branches in the system, including creating new branches, editing existing ones, and handling deleted branches.

Accessing Branch Management

  1. Log in to the system with super administrator credentials
  2. Navigate to the Branch Dashboard
  3. Click on "Branches" in the sidebar menu (only visible to super admins)

Viewing Existing Branches

The main branches page displays a table of all active branches with the following columns:

  • # - Branch ID number
  • Branch Name - The name of the branch
  • Code - Unique identifier for the branch
  • Location - Physical address of the branch
  • City - City where the branch is located
  • Status - Active or inactive status
  • Created At - Date the branch was created
  • Actions - Available actions for the branch

Creating a New Branch

  1. Click the "Add New Branch" button (the button may appear as text, an icon like "+", or similar depending on your interface)
  2. In the branch creation modal that opens, fill in the following details:
    • Name - Enter a unique name for the branch
    • Code - Enter a unique code for the branch
    • Location - Enter the physical address
    • Phone - Contact phone number
    • Email - Contact email address
    • Description - Brief description of the branch
    • Manager - Select the branch manager from the user list
    • Country, State, City - Geographic information
    • Postal Code - ZIP or postal code
    • Timezone - Select the appropriate timezone
    • Status - Toggle to set as active or inactive
  3. Click "Save" to create the branch

Note: Clicking the close button (typically an "X" icon) in the top-right corner of any modal will close it without saving changes.

Editing an Existing Branch

  1. Find the branch you want to edit in the branches table
  2. Click the "Edit" button (the button may appear as text, a pencil icon, or similar depending on your interface) in the Actions column for that branch
  3. The branch editing modal will open with current values pre-filled
  4. Modify the branch information as needed
  5. Click "Save" to update the branch

Deleting a Branch

  1. Find the branch you want to delete in the branches table
  2. Click the "Delete" button (the button may appear as text, a trash icon, or similar depending on your interface) in the Actions column
  3. A confirmation dialog will appear asking if you're sure you want to delete the branch
  4. Click "Confirm" to delete the branch or "Cancel" to keep it

Warning: Deleting a branch moves it to the "Deleted Branches" section. The branch can be restored later if needed.

Managing Deleted Branches

To access deleted branches:

  1. Go back to the main Branch Dashboard menu
  2. Click on "Deleted Branches" in the sidebar menu

Restoring a Deleted Branch

  1. Find the branch you want to restore in the deleted branches table
  2. Click the "Restore" button (the button may appear as text, a refresh icon, or similar depending on your interface) in the Actions column
  3. Confirm the restoration when prompted

Permanently Deleting a Branch

  1. Find the branch you want to permanently delete in the deleted branches table
  2. Click the "Delete Permanently" button (the button may appear as text, a trash icon, or similar depending on your interface) in the Actions column
  3. A confirmation dialog will appear warning that this action cannot be undone
  4. Click "Confirm" to permanently delete the branch or "Cancel" to keep it in the deleted section

Bulk Operations

You can perform actions on multiple branches at once:

  • Select multiple branches using the checkboxes in the first column of the table
  • Choose an action from the "Bulk Actions" dropdown (e.g., "Delete Selected")
  • Confirm the action when prompted

Filtering and Searching Branches

Use the filtering options to find specific branches:

  • Search - Enter text to search in name, code, location, or city
  • Status Filter - Filter by active/inactive status
  • Country Filter - Filter by country
  • City Filter - Filter by city
  • Manager Filter - Filter by assigned manager
  • Date Range - Filter by creation date range
