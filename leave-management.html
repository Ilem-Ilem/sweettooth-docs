Leave Management

The Leave Management system provides comprehensive tools for requesting, approving, tracking, and managing employee leave. This system helps organizations maintain accurate leave records and ensure proper coverage during employee absences.

Overview

The Leave Management system consists of several interconnected modules:

Leave Application

  • Apply Leave - Submit leave requests
  • My Leaves - View your leave history
  • Leave Balance - Check available leave days

Leave Administration

  • Approve Leaves - Review and approve leave requests
  • Leave Types - Configure different leave categories
  • Manage Allocations - Control leave entitlements

Accessing Leave Management

  1. Log in to the system with appropriate credentials
  2. Navigate to the Branch Dashboard
  3. Click on "Leave" in the sidebar menu under the Organization section
  4. Choose the specific leave management function you need

User Roles and Permissions

Different user roles have varying levels of access:

Regular Employees:

  • Can apply for leave
  • Can view their own leave history
  • Can check their leave balance
  • Can cancel their own pending leave applications

Managers/HR Personnel:

  • Can approve or reject leave applications
  • Can view leave applications for their team/department
  • Can manage leave allocations for employees

HR Administrators:

  • Can manage leave types and configurations
  • Can allocate leave days to employees
  • Have full access to all leave management features

Leave Types

The system supports various types of leave with configurable settings:

  • Annual Leave - Standard vacation days
  • Sick Leave - Medical absence
  • Maternity/Paternity Leave - Parental leave
  • Emergency Leave - Unforeseen circumstances
  • Study Leave - Educational purposes
  • Unpaid Leave - Without pay

Leave Allocation System

The system manages leave entitlements with the following features:

  • Default days per year for each leave type
  • Individual employee allocations that can differ from defaults
  • Yearly allocation cycles
  • Tracking of used, remaining, and pending days
