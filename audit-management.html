Audit Management

The Audit Management section allows authorized users to monitor system activities, track changes, and approve or reject requests that require administrative approval.

Accessing Audit Management

  1. Log in to the system with appropriate credentials
  2. Navigate to the Branch Dashboard
  3. Click on "Audit" in the sidebar menu

Audit Management Tabs

The Audit Management page is organized into two main tabs:

  • Logs - View historical audit logs of system activities
  • Approvals - Review and approve/reject pending requests

Audit Logs Tab

This tab displays a chronological record of all system activities:

Information Displayed:

  • User - The user who performed the action
  • Action - The type of action performed (create, update, delete, etc.)
  • Description - Details about what happened
  • Model - The type of record affected
  • Status - The outcome of the action
  • Timestamp - When the action occurred

Filtering Options:

  • Search - Enter text to search in descriptions or actions
  • Department Filter - Filter logs by specific department
  • Action Filter - Filter by specific action types
  • Status Filter - Filter by action status
  • Date Range - Filter by date range

Controls:

  • Sort Columns - Click on column headers to sort the data
  • Pagination Controls - Navigate through multiple pages of logs
  • Clear Filters Button - (the button may appear as text, an icon, or similar depending on your interface) Click to reset all filters

Approvals Tab

This tab displays pending requests that require approval:

Information Displayed:

  • Requester - The user who initiated the request
  • Action - The type of action requested
  • Description - Details about the request
  • Status - Current status (pending, approved, rejected)
  • Created At - When the request was made

Filtering Options:

  • Search - Enter text to search in descriptions or requester names
  • Action Filter - Filter by specific action types
  • Status Filter - Filter by request status
  • Date Range - Filter by date range

Actions Available:

  • Approve Button - (the button may appear as text, an icon, or similar depending on your interface) Click to approve a pending request
  • Reject Button - (the button may appear as text, an icon, or similar depending on your interface) Click to reject a pending request
  • View Details Button - (the button may appear as text, an icon, or similar depending on your interface) Click to see more details about a request

Approving Requests

To approve a pending request:

  1. Navigate to the "Approvals" tab
  2. Locate the request you wish to approve in the table
  3. Click the "Approve" button (the button may appear as text, an icon, or similar depending on your interface) in the Actions column for that request
  4. A confirmation dialog may appear - confirm the action if you wish to proceed
  5. The request status will update to "approved" and the action will be executed

Rejecting Requests

To reject a pending request:

  1. Navigate to the "Approvals" tab
  2. Locate the request you wish to reject in the table
  3. Click the "Reject" button (the button may appear as text, an icon, or similar depending on your interface) in the Actions column for that request
  4. You may be prompted to provide a reason for rejection
  5. Enter your reason and confirm the rejection
  6. The request status will update to "rejected" and the action will not be executed

Inventory Approvals Page

There is also a dedicated page for inventory-related approvals:

Accessing Inventory Approvals:

  1. Navigate to the Branch Dashboard
  2. Go to the "Audit" section
  3. Click on "Inventory Approvals" (this may be a submenu item)

Inventory Approval Types:

  • Stock Adjustments - Approve changes to inventory quantities
  • Item Creation - Approve requests to add new items to inventory
  • Item Updates - Approve changes to existing inventory items
  • Item Deletions - Approve removal of items from inventory
  • Purchase Creation - Approve requests to create new purchases
  • Purchase Deletions - Approve requests to delete purchases

Inventory Approval Actions:

  • Approve Button - (the button may appear as text, an icon, or similar depending on your interface) Click to approve an inventory-related request
  • Reject Button - (the button may appear as text, an icon, or similar depending on your interface) Click to reject an inventory-related request
  • View Details Button - (the button may appear as text, an icon, or similar depending on your interface) Click to see more details about an inventory request
  • Filter Controls - Use filters to narrow down the list of requests

Switching Between Tabs

To switch between different audit management views:

  1. Look for the tab navigation at the top of the audit management page
  2. Click on the tab name you want to view ("Logs" or "Approvals")
  3. The table and filtering options will change to show the appropriate data

Filtering and Searching

Use the filtering options to find specific audit entries or requests:

  • Enter keywords in the search box to find specific entries
  • Use dropdown filters to narrow down by action type, status, or department
  • Set date ranges to focus on specific time periods
  • Click the "Clear Filters" button (the button may appear as text, an icon, or similar depending on your interface) to reset all filters
