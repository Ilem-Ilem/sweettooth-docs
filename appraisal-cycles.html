Appraisal Cycles

The Appraisal Cycles system allows HR administrators to manage performance review periods, schedule appraisals, and organize the performance evaluation process across departments.

Overview

Appraisal Cycles define the periods during which performance evaluations take place:

Cycle Management

Create, edit, and manage appraisal periods

Department-Specific Cycles

Configure cycles for specific departments or organization-wide

Accessing Appraisal Cycles

  1. Log in to the system with HR administrator credentials
  2. Navigate to the Branch Dashboard
  3. Click on "HR" in the sidebar menu under the Organization section
  4. Click on "Appraisal Cycles" in the submenu
  5. The Appraisal Cycles page will display all existing cycles

Appraisal Cycle List

The main page displays all appraisal cycles with their details:

Information Displayed:

  • Cycle Name - Descriptive name for the appraisal cycle
  • Start Date - Date when the cycle begins
  • End Date - Date when the cycle ends
  • Due Date - Deadline for completing appraisals
  • Status - Planned, Active, or Completed
  • Department - Department the cycle applies to (or all departments)
  • Created At - Date the cycle was created

Actions Available:

  • Create Cycle - (the button may appear as text, a plus icon, or similar depending on your interface) Click to create a new appraisal cycle
  • Edit - (the button may appear as text, a pencil icon, or similar depending on your interface) Click to modify an existing cycle
  • Activate - (the button may appear as text, a play icon, or similar depending on your interface) Click to make this the active cycle
  • Complete - (the button may appear as text, a check icon, or similar depending on your interface) Click to mark the cycle as completed
  • Delete - (the button may appear as text, a trash icon, or similar depending on your interface) Click to remove the cycle

Creating an Appraisal Cycle

Create a new appraisal cycle to define a performance review period:

Starting Creation:

  1. Click the "Create Cycle" button (the button may appear as text, a plus icon, or similar depending on your interface) on the main page
  2. A modal or form will appear with cycle creation fields
  3. Fill in the required information for the new cycle

Required Fields:

  • Name - (text input) Enter a descriptive name for the cycle (e.g., "Q1 2026 Annual Review")
  • Start Date - (date picker) Select when the cycle begins (required)
  • End Date - (date picker) Select when the cycle ends (required, must be after start date)
  • Due Date - (date picker) Select the deadline for completing appraisals (required, must be between start and end dates)

Optional Fields:

  • Status - (dropdown) Select the initial status (defaults to "Planned")
  • Department - (dropdown) Select a specific department for this cycle (or leave blank for organization-wide)

Saving the Cycle:

  1. Verify all required fields are completed correctly
  2. Click the "Create Cycle" button (the button may appear as text, an icon, or similar depending on your interface)
  3. The new cycle will be added to the list and available for use

Editing an Appraisal Cycle

Modify an existing appraisal cycle as needed:

Starting Editing:

  1. Find the cycle you want to edit in the list
  2. Click the "Edit" button (the button may appear as text, a pencil icon, or similar depending on your interface) in the Actions column
  3. A modal or form will appear with the current cycle information

Modifiable Fields:

  • All fields from the creation process can be modified except for the cycle ID
  • Be cautious when changing dates for active cycles as it may affect ongoing appraisals

Saving Changes:

  1. Make the necessary changes to the cycle information
  2. Click the "Update Cycle" button (the button may appear as text, an icon, or similar depending on your interface)
  3. The changes will be saved and reflected in the cycle list

Activating an Appraisal Cycle

Set a cycle as active to make it available for conducting appraisals:

Activation Process:

  1. Find the planned cycle you want to activate in the list
  2. Click the "Activate" button (the button may appear as text, a play icon, or similar depending on your interface) in the Actions column
  3. The system will automatically deactivate any other active cycles
  4. The selected cycle will become the active cycle for appraisals

Important Considerations:

  • Only one cycle can be active at a time
  • Activating a new cycle will deactivate the current active cycle
  • Ensure the start date has passed before activating
  • Notify managers and employees about the new active cycle

Completing an Appraisal Cycle

Mark a cycle as completed when all appraisals are finished:

Completion Process:

  1. Find the active or planned cycle you want to complete in the list
  2. Click the "Complete" button (the button may appear as text, a check icon, or similar depending on your interface) in the Actions column
  3. The cycle status will change to "Completed"
  4. The cycle will no longer be available for new appraisals

Post-Completion Actions:

  • Generate reports based on the completed cycle data
  • Archive the cycle if it's no longer needed
  • Begin planning for the next appraisal cycle

Deleting an Appraisal Cycle

Remove an appraisal cycle from the system:

Deletion Process:

  1. Find the cycle you want to delete in the list
  2. Click the "Delete" button (the button may appear as text, a trash icon, or similar depending on your interface) in the Actions column
  3. A confirmation dialog will appear asking if you're sure you want to delete the cycle
  4. Click "Confirm" to delete the cycle or "Cancel" to keep it

Important Considerations:

  • Only cycles without completed appraisals should be deleted
  • Deleting a cycle removes it permanently from the system
  • Completed appraisals tied to the cycle may become inaccessible
  • Consider completing the cycle instead of deleting if appraisals have started

Filtering and Searching

Use the filtering options to find specific appraisal cycles:

Search Options:

  • Search Box - (text input) Enter text to search across cycle names

Filter Options:

  • Status Filter - (dropdown) Filter by cycle status (all, planned, active, completed)

Cycle Statuses

Appraisal cycles have different statuses that determine their availability:

Status Definitions:

  • Planned - Cycle is scheduled but not yet active; appraisals cannot be conducted
  • Active - Cycle is currently accepting appraisals; only one cycle can be active at a time
  • Completed - Cycle has finished; no new appraisals can be started, but existing ones remain accessible

Status Transitions:

  • Planned → Active (when activated)
  • Active → Completed (when marked as completed)
  • Planned → Completed (direct transition possible if needed)

Department-Specific Cycles

Create appraisal cycles for specific departments:

Creating Department-Specific Cycles:

  1. When creating a cycle, select a specific department from the department dropdown
  2. The cycle will only be available to employees in that department
  3. Employees in other departments will not see this cycle during appraisals

Organization-Wide Cycles:

  1. Leave the department field blank when creating a cycle
  2. The cycle will be available to all employees in the organization
  3. This is the default behavior when no department is specified

Tips and Best Practices

  • Plan appraisal cycles well in advance to allow adequate preparation time
  • Coordinate with department heads to ensure alignment with business cycles
  • Communicate cycle schedules clearly to all managers and employees
  • Set realistic due dates that allow sufficient time for thorough appraisals
  • Consider seasonal business fluctuations when scheduling cycles
  • Use department-specific cycles for teams with different performance rhythms
  • Regularly review and update cycle schedules based on organizational changes
  • Archive completed cycles after generating necessary reports
  • Ensure HR staff are trained on cycle management procedures
